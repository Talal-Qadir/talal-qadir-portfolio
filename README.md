# 🚀 Talal Qadir Portfolio (Full Stack)

This is a full-stack portfolio web application built using **Django (Backend API)** and **React (Frontend with Vite)**. It showcases personal projects, skills, and includes a contact system using REST APIs.

## ⚙️ Tech Stack

Frontend: React (Vite), JavaScript, Axios, CSS/Tailwind  
Backend: Django, Django REST Framework  
Database: SQLite (development) / PostgreSQL (production recommended)  
Deployment: Vercel (Frontend), Render/Railway (Backend recommended)

---

## 📁 Project Structure

talal-qadir-portfolio/
├── backend/ (Django API)
├── frontend/ (React app)

---

## 🚀 Features

- Personal portfolio website
- Projects fetched from Django API
- Contact form with backend integration
- Responsive modern UI
- REST API based architecture
- Fast frontend performance
